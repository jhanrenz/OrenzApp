# Sales Management SaaS — Business Operations & Analytics System

## Description

A full-stack SaaS application designed to manage and optimize business sales operations. The system provides tools for product and menu management, cost tracking, and profit analysis, along with a real-time dashboard to monitor daily business performance.

Built with a focus on simplifying workflows and improving decision-making through data visibility. The platform is actively maintained, with continuous feature enhancements and ongoing UI improvements.

## Tech Stack

**Backend:**

* Laravel (REST API)
* Laravel Sanctum (Authentication)

**Frontend:**

* React (Functional Components & Hooks)
* TypeScript
* TanStack Query (Server State Management)
* Axios

## Core Features

* Product and menu management
* Cost and profit tracking
* Sales monitoring dashboard
* Authentication & authorization
* API-driven architecture

## Status

* Actively maintained
* Continuous UI/UX improvements
